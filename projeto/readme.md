# FastAPI

Para rodar o projeto, crie a virtual env
```bash
python3 -m venv venv
```

Ative a virtual env
```bash
source venv/bin/activate
```

Instale as dependencias
```bash
pip install -r requirements.txt
```

Rode o projeto
```bash
uvicorn main:app --reload
```

Acesse o projeto em http://127.0.0.1:8000
